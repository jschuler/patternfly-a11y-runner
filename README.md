# patternfly-a11y runner
Minimal dependencies setup to run the patternfly-a11y tool on any domain

## Install
```
git clone https://github.com/jschuler/patternfly-a11y-runner.git
yarn install
```

## Run
`yarn a11y --help`

## View report
`yarn serve`
